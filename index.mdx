OpenAPI Generator allows generation of API client libraries (SDK generation), server stubs, documentation, and configuration automatically given an [OpenAPI Spec](https://github.com/OAI/OpenAPI-Specification) (v2 and v3). It supports over 50 languages and frameworks — from Java and TypeScript to Go, Python, Kotlin, Rust, and many more.

## Get started in 3 steps

<Steps>
  <Step title="Install OpenAPI Generator">
    Install using your preferred method:

    ```bash npm
    npm install @openapitools/openapi-generator-cli -g
    ```

    ```bash Homebrew
    brew install openapi-generator
    ```

    ```bash Docker
    docker pull openapitools/openapi-generator-cli
    ```
  </Step>
  <Step title="Point it at your OpenAPI spec">
    Use any valid OpenAPI 2.0 or 3.0 specification — a local file, a URL, or the Petstore example to get started immediately.
  </Step>
  <Step title="Generate and use your code">
    Run the `generate` command with your chosen generator and output directory. The generated code is immediately usable.

    ```bash
    openapi-generator-cli generate \
      -i petstore.yaml \
      -g typescript-axios \
      -o ./generated-client
    ```
  </Step>
</Steps>
